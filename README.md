📝 Notes Management System

A full-stack web application built with Flask that enables users to securely create, manage, search, update, and delete personal notes. The application features user authentication, session management, database integration, and a responsive Bootstrap-based interface..
This project demonstrates practical implementation of CRUD operations, user authentication, frontend-backend integration, and database management while following Flask development best practices. It was developed to strengthen skills in backend development, web application architecture, and real-world problem solving.

 ✨ Key Features
* User Registration and Login
* Secure Session Management
* Create, Read, Update, and Delete Notes
* Note Search Functionality
* Responsive Bootstrap UI
* SQLite Database Integration
* Flask Template Rendering with Jinja2

 🛠️ Tech Stack
Python • Flask • SQLite • HTML • CSS • Bootstrap • Jinja2 • Git • GitHub

🎯 Skills Demonstrated
* Backend Development
* Database Design and Management
* Authentication and Authorization
* Session Handling
* CRUD Operations
* Frontend-Backend Integration
* Debugging and Problem Solving
